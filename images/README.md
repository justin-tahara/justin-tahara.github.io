# Images Directory

Add your images here:

- **profile.jpg** - Your professional headshot or profile photo (recommended size: 800x800px)
- Add any other images you want to use on your website

## Recommended Images

1. **profile.jpg** - Main profile picture (circular crop recommended)
2. Photography portfolio images (if you want to showcase your work)
3. Company logos for experience section (optional)

## Image Optimization Tips

- Use JPEG for photos (better compression)
- Use PNG for logos or images with transparency
- Recommended maximum width: 1920px for full-width images
- Compress images before uploading (use tools like TinyPNG or ImageOptim)
